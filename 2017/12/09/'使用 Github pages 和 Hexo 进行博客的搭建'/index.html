使用 Github pages 和 Hexo 进行博客的搭建

发表于 2017-12-09

我们需要用使用Github Pages和Hexo。
使用Github Pages需要这个服务需要拥有一个Github账号，你需要注册一个。
使用Hexo用到一个分布式版本控制软件Git用来与Github进行通讯和一个JavaScript运行环境Node.js用来进行本地测试。
Github是一个代码仓库可以帮助我们方便的同步版本和对版本进行分支修改，是世界上最大的代码存放网站和开源社区。
Hexo是一个快速、简洁且高效的博客框架。Hexo使用Markdown(或其他渲染引擎)解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

开启GitHub Pages

需要去Github进行注册，之后”New repository”

需要注意”Repository name”需要用”XX.github.io”这种格式来命名。
“Create repository”之后会有一个”repository”的地址。这个需要记录下来，为以后和这个”repository”进行通信使用。

最后在”repository”的”Settings”里找到”Github Pages”按下”Choose a theme”自己喜欢那个挑那个吧！选好之后再到”Setting”里的”GitHub Pages”里可以得到一个地址，这个地址就是你博客的地址了，现在这里应该有一个你选择的主题。
